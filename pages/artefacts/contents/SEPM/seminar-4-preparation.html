Data Structures

You will now become familiar with data structures, which are approaches to organise, store and manage electronic data. To do this, visit python.org (n.d.), and any other relevant sources of information that you may find useful to support this task.

After visited the python.org website, I selected the list which is one of built-in sequence types, and the dict which is a built-in mapping types in Python for holding the data associated with the list of functional and non-functional requirements for the assignment.

The list data structure can hold images resources for the character instance. It allows the character instance to access the defined images through the interaction for-loop. Moreover, the dict type can hold the definition of animations for each character instance. It is a key-value structure. So, it allows to get the value by using the key with O(1) time complexity.

I think the work of David Brayford (2019) is similar to the work of Abeykoon et al. (2020). We can see that both authors were going to explore the way of securing HPC systems. They introduced existing methods, and conducted empirical experiments to prove their hypothesis.
